# CN Form Faker

CN Form Faker 是一款 Chrome 扩展，用于快速为表单填充中国本地化测试数据。

它可以识别常见的输入框、文本域、下拉选择、组件库自定义选择器、弹窗和必填字段，并生成看起来真实的中文姓名、手机号、身份证号、公司名称、地址、日期、金额以及其他测试值。

## 功能

- 通过扩展弹窗填充当前弹窗或表单。
- 在测试较长流程时填充整个页面。
- 需要生成全新测试记录时，可覆盖已有值。
- 通过右键菜单按指定字段类型填充。
- 按住 Alt 并双击字段，可直接填充该字段。
- 支持多种常见 UI 库，包括 Ant Design、Element Plus、Semi Design、Arco Design、MUI、Layui，以及 EasyUI 风格的组合框。

## 本地安装

1. 打开 `chrome://extensions`。
2. 启用开发者模式。
3. 点击“加载已解压的扩展程序”。
4. 选择当前文件夹。

## 项目名称

项目名称为 **CN Form Faker**，强调它面向中国本地化表单测试数据，同时保持简短、适合 GitHub 使用。
