# CN Form Faker

CN Form Faker is a Chrome extension for quickly filling forms with China-localized test data.

It can detect common inputs, textareas, selects, custom component selects, dialogs, and required fields, then generate realistic-looking Chinese names, mobile numbers, ID card numbers, companies, addresses, dates, amounts, and other test values.

## Features

- Fill the current dialog or form from the extension popup.
- Fill the whole page when testing larger flows.
- Overwrite existing values when you need a fresh test record.
- Use the context menu to fill a specific field type.
- Alt-double-click a field to fill it directly.
- Supports many common UI libraries, including Ant Design, Element Plus, Semi Design, Arco Design, MUI, Layui, and EasyUI-style comboboxes.

## Install Locally

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click Load unpacked.
4. Select this folder.

## Project Name

The project name is **CN Form Faker**, emphasizing China-localized form testing data in a short GitHub-friendly name.
